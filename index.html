Serving rhythm

Light reds and fuller whites show best with a little air but no theatre. Mature reds deserve gentle handling; structured bottles can sit in the decanter while food comes together.

Storage cues

Use the storage field on every bottle card. Fridge bottles are ready cold; fridge-before-serving reds need a short chill to land at a polished restaurant temperature.

Food matching

Pairings are searchable. Type the dish, choose a quick pick, or open any bottle for the full pairing list and guest-facing tasting note.

Stock control

Bottle count changes are saved on this iPad. Use the editor to export a fresh JSON file when you want the published list to match the latest cellar count.

Admin-friendly workflow

Edit the JSON below, import a replacement file, or adjust bottle counts from detail pages. Download the updated file as wines.json, replace the file in this folder, then redeploy to Vercel or Netlify.

Data checklist

  • Keep every wine as one JSON object.
  • Required fields include name, type, country, region, grapes, score, storage, drink, status, serve, notes, pairings, and bottles.
  • Pairings must stay as an array of text values.
